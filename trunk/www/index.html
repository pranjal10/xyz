Unified Expression Language

This project provides an implementation of the unified expression language (UEL) 1.1.  The main goals are

  • Improves current implementation: bug fixes and performance improvements
  • Provides API for use by other tools, such as Netbeans

Learn about Expression Language 1.1

Download
Download the latest EL implementation and run with a JavaEE 5 container.

Get Involved
Help improve EL implementation by providing feedbacks and filing bugs. Those interested in code contribution, can set up a workspace for building and testing.
